# Доска объявлений. Интернет-ресурс для фанатского сервера одной MMORPG.


## Описание
Данный сервер предназначен для отслеживанья новостей в MMORPG.
- Регестрация;
- Получения письма с кодом подтверждения входа;
- После регестрации становится доступно создание и редоктирования объявления, а также отправлять комментарии на чужие объвления.
- Не зарегистрированный пользователь имеет возможность смотреть объявлеения (посты) и фильтровать их.
- Сами объявления могут содержать: заголовок, текст внутри которого могут находится картинки.
- При отправки комментария пользователь отправивший комментарий получит письмо на почту о том что он отправил комментарий и ссылку на объявления. Пользователь, создавший объявления получит письмо, о том что на его объявлении комментарий и ссылку на объявления.
- Зарегистрированному пользователю доступна страница с комментариями оставленными к его постам и возможность их сортировка
- Возможность подписываться на конкретную категорию постов, при появлении нового поста в любимых категориях на почту приходит уведомление: "Новый пост в любимой категории и ссылка на него".
- Рассылка каждый понедельник со всеми постами созданными за последнюю неделю в любимых категориях пользователя и ссылками на них.

## Установка и настройка

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/Dushenka525/SiteMMO.git
   cd your-repo


2. **Создать и клонировать git репозиторий**
python -m venv venv
source venv/bin/activate      # для Linux/Mac
venv\Scripts\activate          # для Windows

3. **Установить зависимости**
pip install -r requirements.txt

4. **Настроить переменные окружения**
Создать файл .env в корне проекта (рядом с manage.py) со следующим содержимым:

SECRET_KEY = your_secret_key
EMAIL_HOST_PASSWORD = your_host_password  # пароль от почты

EMAIL_HOST_USER = your_host_user
DEFAULT_FROM_EMAIL = your_default_from_email

5. **Выполнить миграции**
python manage.py migrate

6. **Запустить сервер разработки**
python manage.py runserver
